The Brief
Signal
Sticker booths and fake merch turn identity into a composition game instead of a finished graphic.
Archive Link
The toy continues the collectible language of Joydotz, Wish Quilt, keychains, patches, and grillz through interaction.
Interaction
A single drop creates momentum; dragging, rotating, scaling, and deleting let the user recover authorship.
Output
The end state is not just a webpage. It is a saved PNG artifact ready to keep, post, or remix.
Design Decisions
Arcade cabinet, not editor chrome
The bold marquee and physical button hierarchy make the tool feel like a booth you discover, while familiar controls keep the composition workflow immediately understandable.
Random first, manual second
The drop creates a useful surprise with varied position, rotation, scale, and stacking. Direct manipulation follows so the output feels authored rather than generated and abandoned.
One palette, three moods
Baby pink, chrome black, and neon green change the booth and product surface together. The presets are distinct enough to shift the object without creating a dense styling panel.
Transparent defaults
The supplied visual references are flattened JPGs with baked-in white backgrounds, so the default set uses CSS and text-based objects that keep clean sticker edges. User-uploaded PNG, JPG, and WebP files remain supported.
